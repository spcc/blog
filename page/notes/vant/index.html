# vant实现移动端的适配

  1. 安装amfe-flexible和postcss-pxtorem
npm i -S amfe-flexible
npm install postcss-pxtorem --save-dev
1
2
  1. 在main.js文件中引入lib-flexible
import 'amfe-flexible/index.js'
1
  1. 项目配置postcss 在/目录新建.postcssrc.js
module.exports = {
  plugins: {
    autoprefixer: {
      browsers: ['Android >= 4.0', 'iOS >= 8']
    },
    'postcss-pxtorem': {
      rootValue: 37.5,
      propList: ['*']
    }
  }
}

需注意

  1. css里面只需要写px，然后会自动转换成rem在浏览器显示
  2. rootValue设置为37.5，是为了引用像vant这样的第三方UI框架，因为第三方框架没有兼容rem，用的是px单位，将rootValue的值设置为设计图宽度（这里为750px）75的一半，即可以1:1还原vant、mint-ui的组件，否则会样式会有变化，例如按钮会变小。
  3. 既然设置成了37.5 那在写样式时，也将值改为设计图的一半。
